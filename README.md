# 1. Lancer le TP

## TP4 :

- Préparer le terraform :
```
terraform init
```

- Crée la VM :
```
terraform apply
```

- Accéder à la VM :
```
ssh -i id_rsa devops@{adresse IP publique} 
```

- Détruire ce qui a été fait :
```
terraform destroy
```
